Financial Formulas & Analysis

Return on Equity (ROE):

$$ROE = \frac{Net\ Income}{Shareholder\ Equity} = \frac{28.6M}{125M} = 22.88\%$$

Earnings Per Share (EPS):

$$EPS = \frac{Net\ Income - Preferred\ Dividends}{Weighted\ Average\ Shares} = \frac{28.6M - 0}{12.2M} = \$2.34$$

Price-to-Earnings Ratio (P/E):

$$P/E = \frac{Market\ Price\ per\ Share}{Earnings\ Per\ Share} = \frac{42.50}{2.34} = 18.16$$

These are the key financial ratios investors care about. Our ROE of 22.88% is above industry average. P/E ratio of 18.16 shows we're fairly valued relative to earnings.

Portfolio Optimization Code


// Portfolio risk calculation
function calculatePortfolioRisk(weights, covMatrix) {
    let variance = 0;

    for (let i = 0; i < weights.length; i++) {
        for (let j = 0; j < weights.length; j++) {
            variance += weights[i] * weights[j] * covMatrix[i][j];
        }
    }

    return Math.sqrt(variance);
}

// Example usage
const weights = [0.4, 0.35, 0.25];
